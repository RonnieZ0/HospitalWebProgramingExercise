<!-- ABOUT THE PROJECT -->
## About The Project

This is a a Hospital Patient Portal that allows users to add and delet patient name from the hostital's database

<p align="right">(<a href="#top">back to top</a>)</p>



### Built With

* [Flask](https://flask.palletsprojects.com)
* [Bootstrap](https://getbootstrap.com)
* [SQLite](https://www.sqlite.org)

<p align="right">(<a href="#top">back to top</a>)</p>



<!-- GETTING STARTED -->
## Getting Started

Please Download the code and run it on your local machine.

### Prerequisites

* Flask
* SQLite


<p align="right">(<a href="#top">back to top</a>)</p>



<!-- USAGE EXAMPLES -->
## Usage

The administraters of a hospital can use this portal to manage the patient wit their names. Each of the patient can be added and deleted from the database. The users can view the current table in the database on 'Delete Patient Name' page.

_For more examples, please refer to the [Documentation](https://example.com)_
